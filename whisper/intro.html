Whisper - Clean, Responsive, Easy To Customize WordPress Theme

Whisper is a multi-featured, SEO-optimized WordPress theme with modern and clean design. Built with HTML5 and CSS3 (LESS), Whisper is 100% responsive and easy to customize via powerful options. This theme is very intuitive to use and completely ready to operate out of the box. The flexibility of the theme makes Whisper can be used for any type of websites: business, corporation, blog, personal, portfolio, etc.

We made Whisper with care of usability, user experience and ability to customize. With creative theme options page as well as very detailed instructions and documentation, users will take less than 10 minutes to setup and use Whisper. We promise to constantly improve our product, add more requested features from users, care about support as well as provide lifetime updates!

Whisper has a lot of awesome features such as: fully responsive, shortcode generator, creative theme options, boxed and wide layout style, full translation support, 4 predefined color schemes, advanced portfolio and blog options, custom widgets, icon font integration, advanced typography options, advanced background image options, 20+ PSDs included, custom CSS field, SEO optimized and extensive documentation! Whisper Highlighted Features

Whisper Full Feature List:

Highlights:

  • SEO Optimized: uses semantic HTML5 tags, auto generates meta description and meta keywords, optimizes page title, supports microdata. Compatible with popular SEO plugins (WordPress SEO by Yoast, AIO SEO, Platinum SEO Pack, HeadSpace, wpSEO).
  • Creative Theme Options: unique, creative and easy-to-use theme options page that help you to customize any part of your site quickly and easily!
  • Advanced Shortcode Generator: intuitive shortcode generator with amazing user interface. With 20+ shortcodes: buttons, recent posts, recent portfolios, boxes, icons, tabs, toggles, maps, dividers, column, promotion  box, etc., you can built custom pages layout / content with ease!
  • Custom Post/Page Options: customize featured title area, slider, sidebar layout for specific posts and pages.
  • Post Formats Supported: change how each post looks by choosing a post format. Supports formats: standard, image, gallery, video, audio, quote, link.

Design:

  • Clean and modern design that can be used for any type of website: business, corporation, personal blog, portfolio, etc.
  • Support touch icon for mobile devices and tablets
  • Custom logo upload
  • Fully responsive
  • Built with HTML5 and CSS3, compatible with all modern browsers (IE 8+, Firefox, Google Chrome, Safari, Opera)
  • Modular CSS using LESS: LESS is a modern way to code CSS, which keep the code well organized and optimized
  • Boxed and wide layout styles
  • 4 predefined color schemes: red, orange, blue and green
  • 3 predefined homepage layouts: users still can create custom page (homepage) layouts with help of powerful shortcodes
  • 2 blog layouts: normal and boxes
  • 2 portfolios layouts: normal and hexagon
  • 2 single portfolio layouts: with Nivo slider and simple
  • 5 predefined background patterns or use custom background
  • Custom sidebar layout (no sidebar, left sidebar, right sidebar) for whole website, blog posts and pages
  • Easy to customize featured title area: show/hide breadcrumbs, upload custom background
  • Easy to customize typography (font family, font size, line height, font style and color) for body text and all headings. Support Google fonts and standard fonts.
  • 140+ icon fonts and 60 social icon fonts
  • Custom footer layout: you can choose how many columns in footer and drag and drop widgets for each column
  • Custom footer text: display copyright text and contact information with ease using theme options and built-in shortcodes
  • Custom CSS supported

Advanced Post/Page Options:

  • Show/hide Featured Title Area
  • Custom title shown in Featured Title Area
  • Subtitle in Featured Title Area
  • Custom background for Featured Tittle Area
  • Show/hide breadcrumbs
  • Custom slider: support Revolution slider and any slider that has shortcode
  • Custom layout style: boxed or wide
  • Custom sidebar layout: no sidebar, left sidebar and right sidebar
  • Enable/disable page comments

Advanced Blog Options:

  • 2 blog layouts: normal and boxes
  • 7 post formats supported: standard, image, gallery, video, audio, quote, link.
  • Threaded comments
  • Select which type of post content will be displayed on blog page: post excerpt, post content or post content before more tag
  • Set content limit for blog posts
  • Customize read more text
  • And all options from custom post/page options above

Custom page templates:

  • Blog: template to display blog posts
  • Blox Boxes: display blog posts in boxed style
  • Contact: template for contact page, including Google Maps and other options when edit page
  • Portfolio: display portfolios
  • Portfolio Hexagon: display portfolios in creative hexagon style

Custom widgets:

  • Tweets widget: display recent tweets from a twitter account. Compatible with latest Twitter API.
  • Recent Posts widget: display recent posts from your blog with many options to show/hide thumbnail, post date, comments, excerpt.
  • Tabs widget: display recent posts and recent comments in a beautiful tab widget.
  • Social Feed widget: display recent items from the feed of creative social networks: Pinterest, Deviant Art, Flickr, Dribbble, Youtube, Instagram and Newsfeed (RSS).
  • Social Media widget: display links to social media network profiles

Supported Plugins:

  • Revolution Slider Plugin (included with theme) as well as other slider plugins such as Meteor Slider, Soliloquy Slider
  • Popular SEO plugins: WordPress SEO by Yoast, AIO SEO, Platinum SEO Pack, HeadSpace, wpSEO
  • Pricing Table plugin: create pricing table easily with intuitive user interface, no shortcodes needed. Whisper already customized the built-in styles of the plugin to match theme design.
  • Other popular plugins that can't be listed all here

General:

  • WordPress Multi-Site (WPMS) compatible
  • Translation ready
  • HTML version and 23 layered PSDs included for customization
  • Demo content included: help you to setup a demo website less than 10 minutes
  • Child theme compatible
