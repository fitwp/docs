<h2>Whisper - Clean, Responsive, Easy To Customize WordPress Theme</h2>

<p><a href="http://fitwp.com/blog/whisper-2-0-released/" title="What's NEW in Whisper 2.0"><img src="http://fitwp.github.io/docs/whisper/v20.png" alt="whisper 2.0 new features"></a></p>

<p>Whisper is a <strong>multi-featured</strong>, <strong>SEO-optimized</strong> WordPress theme with <strong>modern and clean design</strong>. Built with HTML5 and CSS3 (LESS), Whisper is <strong>100% responsive</strong> and <strong>easy to customize</strong> via powerful options. This theme is very intuitive to use and completely ready to operate out of the box. The flexibility of the theme makes Whisper can be used for any type of websites: business, corporation, blog, personal, portfolio, etc.</p>

<p>We made Whisper with care of usability, user experience and ability to customize. With <a href="http://themes.fitwp.com/whisper/features/creative-theme-options/">creative theme options</a> page as well as very detailed instructions and documentation, users will take <strong>less than 10 minutes to setup and use Whisper</strong>. We promise to constantly improve our product, add more requested features from users, care about support as well as provide <strong>lifetime updates</strong>!</p>

<p>Whisper has a lot of awesome features such as: fully responsive, <a href="http://themes.fitwp.com/whisper/features/shortcode-generator/">shortcode generator</a>, <a href="http://themes.fitwp.com/whisper/features/creative-theme-options/">creative theme options</a>, <a href="http://themes.fitwp.com/whisper/features/boxed-wide/">boxed and wide layout style</a>, full translation support, <a href="http://themes.fitwp.com/whisper/features/4-color-schemes/">UNLIMITED color schemes</a>, advanced portfolio and blog options, custom widgets, icon font integration, advanced typography options, advanced background image options, 20+ PSDs included, custom CSS field, <a href="http://themes.fitwp.com/whisper/features/microformats-microdata/">SEO optimized</a> and extensive documentation! Check them out below!</p>

<h4>Highlights:</h4>
<ul>
	<li><a href="http://themes.fitwp.com/whisper/features/microformats-microdata/"><strong>SEO Optimized</strong></a>: uses semantic HTML5 tags, auto generates meta description and meta keywords, optimizes page title, supports <a href="http://themes.fitwp.com/whisper/features/microformats-microdata/">microdata</a>. Compatible with popular SEO plugins (WordPress SEO by Yoast, AIO SEO, Platinum SEO Pack, HeadSpace, wpSEO).</li>
	<li><strong><a href="http://themes.fitwp.com/whisper/features/creative-theme-options/">Creative Theme Options</a></strong>: unique, creative and easy-to-use theme options page that help you to customize any part of your site quickly and easily!</li>
	<li><strong><a href="http://themes.fitwp.com/whisper/features/customizer/">Customizer</a></strong>: change theme design and preview changes in <strong>real time</strong>.</li>
	<li><strong><a href="http://themes.fitwp.com/whisper/features/shortcode-generator/">Advanced Shortcode Generator</a>: </strong>intuitive shortcode generator with amazing user interface. With 20+ shortcodes: buttons, recent posts, recent portfolios, boxes, icons, tabs, toggles, maps, dividers, column, promotion  box, etc., you can built custom pages layout / content with ease!</li>
	<li><strong><a href="http://themes.fitwp.com/whisper/features/custom-page-layout/">Custom Post/Page Options</a></strong>: customize featured title area, slider, sidebar layout for specific posts and pages.</li>
	<li><a href="http://themes.fitwp.com/whisper/features/post-formats/"><strong>Post Formats Supported</strong></a>: change how each post looks by choosing a post format. Supports formats: standard, image, gallery, video, audio, quote, link.</li>
</ul>
